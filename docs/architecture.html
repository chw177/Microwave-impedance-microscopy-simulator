Circuit structure and its code mapping

The MIM reflectometer as three arms tapped from one source (cf. paper Fig. 1a). Excitation drives the tip; cancellation (voltage-controlled attenuator A + phase shifter ϕ) synthesizes a wave that meets the reflected wave at the combiner and nulls the carrier before amplification; reference (phase shifter ϕ) sets the LO phase for the IQ mixer.
